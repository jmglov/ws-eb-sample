# ws-eb-sample

Sample websocket server on AWS Elastic Beanstalk

## Usage

### Running locally

```
lein run
```

Connect to ws://localhost:8080/ws with a websocket client (try the excellent
[Simple WebSocket Client] [1] Chrome extension).
